# LocalEats - BDD com Pytest-BDD e Playwright

Projeto desenvolvido para a atividade PBL da Aula 12.

---

# Tecnologias utilizadas

- Python
- Pytest
- Pytest-BDD
- Playwright

---

# Instalação

## Criar ambiente virtual

```bash
python -m venv venv
```

## Ativar ambiente virtual

### Windows

```bash
venv\Scripts\activate
```

---

# Instalar dependências

```bash
pip install -r requirements.txt
```

---

# Instalar navegadores do Playwright

```bash
playwright install
```

---

# Executar testes

```bash
pytest -v
```
